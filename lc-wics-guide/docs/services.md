The chapter uses the following services to manage chapter communication and business.

### <span style="color: orange">LastPass</span>

**Use**: Stores logins and passwords for services used by the chapter </br> 
**Login page**: [https://lastpass.com/?ac=1&lpnorefresh=1](https://lastpass.com/?ac=1&lpnorefresh=1) </br>
**Authentication**: Provided by current or previous board members

---

### <span style="color: orange">Gmail</span>

**Use**: General email client for communication </br>
**Authentication**: Available via LastPass

---

### <span style="color: orange">Mailchimp</span>

**Use**: Distribute e-mail to club members regarding announcements, events, newsletters, etc. </br>
**Login page**: [https://us12.admin.mailchimp.com/campaigns/](https://us12.admin.mailchimp.com/campaigns/) </br>
**Authentication**: Available via LastPass

---

### <span style="color: orange">Zapier</span>

**Use**: Connector to trigger acctions between services (like add a new member to mailchimp list from a Google form) </br>
**Login page**: [https://zapier.com/app/home](https://zapier.com/app/home) </br>
**Authentication**: Use Gmail login

---

### <span style="color: orange">Github/Github Pages</span>

**Use**: Hosts the website for the chapter </br>
**Login page**: [https://github.io](https://github.io) </br>
**Authentication**: Available via LastPass

---

### <span style="color: orange">Forestry.io</span>

**Use**: Add, edit blog posts on the Lehman WiCS website </br>
**Login page**: https://app.forestry.io/dashboard/#/</br>
**Authentication**: Use Lehman WiCS Github login

---

### <span style="color: orange">SmallChat</span>

**Use**: Chat bot for communicating on the Lehman WiCS website </br>
**Login page**: [https://small.chat/](https://small.chat/) </br>
**Authentication**: Connected to Slack login

---

### <span style="color: orange">Canva</span>

**Use**: Create and edit flyers for events </br>
**Login page**: [https://www.canva.com/](https://www.canva.com/) </br>
**Authentication**: Use Gmail login

---

### <span style="color: orange">Bit.ly</span>

**Use**: Shorten links to add to communications and flyers; easier to remember </br>
**Login page**: [https://bitly.com/a/sign_in](https://bitly.com/a/sign_in) </br>
**Authentication**: Available via LastPass


---

### <span style="color: orange">Social Media Accounts</span>

All logins for social media accounts can be accessed via LastPass with the exception of Facebook which requires an admin to add you to the FB page.

---

**Every new service used must be added to LastPass to maintain accurate accounts and credentials.**
