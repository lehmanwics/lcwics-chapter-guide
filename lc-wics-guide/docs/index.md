# Start Here

This guide serves as living documentation for how to run events, send emails, accept sponsor funds, etc. Anything you need to know about the club and how to run it should be in this guide.

Use the search bar on the left to search through documentation. 

:exclamation: **IMPORTANT:** When updating this guide, **do not** add any passwords or sensitive information. If you need to reference something in our Google Drive or authentication to a website or service, include a note to go to the specified location where that information is stored. For example, add: "*search Google Drive for 'events'" instead of linking the actual document you are referencing.

*Updated: 9/12/18*

---

This guide uses MkDocs to create documentation. For full documentation visit [mkdocs.org](https://mkdocs.org).

## General MkDocs Commands 

* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs help` - Print this help message.
* `mkdocs gh-deploy` - Deploy changes to Github.
