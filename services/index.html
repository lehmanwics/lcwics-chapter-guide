The chapter uses the following services to manage chapter communication and business.

LastPass

Use: Stores logins and passwords for services used by the chapter
Login page: https://lastpass.com/?ac=1&lpnorefresh=1
Authentication: Provided by current or previous board members

Gmail

Use: General email client for communication
Authentication: Available via LastPass

Mailchimp

Use: Distribute e-mail to club members regarding announcements, events, newsletters, etc.
Login page: https://us12.admin.mailchimp.com/campaigns/
Authentication: Available via LastPass

Zapier

Use: Connector to trigger acctions between services (like add a new member to mailchimp list from a Google form)
Login page: https://zapier.com/app/home
Authentication: Use Gmail login

Github/Github Pages

Use: Hosts the website for the chapter
Login page: https://github.io
Authentication: Available via LastPass

Forestry.io

Use: Add, edit blog posts on the Lehman WiCS website
Login page: https://app.forestry.io/dashboard/#/
Authentication: Use Lehman WiCS Github login

SmallChat

Use: Chat bot for communicating on the Lehman WiCS website
Login page: https://small.chat/
Authentication: Connected to Slack login

Canva

Use: Create and edit flyers for events
Login page: https://www.canva.com/
Authentication: Use Gmail login

Bit.ly

Use: Shorten links to add to communications and flyers; easier to remember
Login page: https://bitly.com/a/sign_in
Authentication: Available via LastPass

Social Media Accounts

All logins for social media accounts can be accessed via LastPass with the exception of Facebook which requires an admin to add you to the FB page.

Every new service used must be added to LastPass to maintain accurate accounts and credentials.
